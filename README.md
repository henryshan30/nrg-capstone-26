# nrg-capstone-26

Mock Microsoft Copilot-style chatbot built with a React frontend and a TypeScript Express backend.

## Run locally

```bash
npm install
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:5050
- Health check: http://localhost:5050/api/health

The backend returns deterministic mock responses. It does not call Microsoft Copilot or any external AI service.
